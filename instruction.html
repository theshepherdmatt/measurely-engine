How to run a sweep in REW

Step-by-step instructions for capturing the perfect measurement — pick your workflow below.

What are we actually measuring?

Measurely isn't testing your speakers — it measures the acoustic response of your room at exactly the position your speakers and listening chair are placed.

You don't just hear the direct sound from the drivers. You hear a complex mix of energy reflecting off every wall, floor, and ceiling surface, stacking up in corners and building resonances that can mask or exaggerate entire frequency bands. The measurement microphone captures all of that — the total acoustic signature of the room precisely as your ears experience it in that chair.

1 Enable the REW API

REW 5.40 includes a local HTTP API that Measurely uses to trigger sweeps and pull your data automatically. You need to start it once.

In REW, go to Help → Enable API (or launch REW from the command line with the -api flag). REW will start listening on port 4735.

How to check: Open the Measurely dashboard. If the Sync from REW button appears in the sidebar, the API is running and Measurely has found it. If it doesn't appear, REW's API isn't active yet.

REW 5.40 or later REW running on the same machine as your browser API enabled in REW Help menu

2 Physical Setup

Place your measurement microphone (e.g. a UMIK-1) exactly at your main listening position — right where the centre of your head is when you sit in the chair.

Crucial: The microphone capsule must point dead straight up towards the ceiling, not forward at the speakers. Pointing it upward captures the full room energy — reflections from all surfaces — which is precisely what Measurely analyses.

Make sure your audio interface is set as REW's input and output device before continuing.

3 Click "Sync from REW" in the Dashboard

Back in the Measurely dashboard, click the Sync from REW button in the sidebar. Measurely will:

  1. Tell REW to run a 20 Hz – 20 kHz sine sweep at 512k length
  2. Wait for REW to finish measuring
  3. Download the impulse response directly from REW's API
  4. Run the full acoustic analysis and update your dashboard automatically

Step away from the mic. Once you click the button, REW will begin playing the sweep signal immediately. Move away from the microphone and stay quiet — footsteps, voices, and HVAC noise will pollute the measurement.

The whole process takes around 30–60 seconds. Your scores and analysis will update as soon as REW finishes.

2 REW Configuration

Open Room EQ Wizard (REW) and click Measure in the top-left corner.

Set your sweep parameters:

  • Type: Sine sweep
  • Range: 20 Hz to 20,000 Hz
  • Length: 256k or 512k

Play both speakers together. In the Output dropdown, select L+R or configure your audio interface to route the sweep to both channels simultaneously. Measurely analyses how your speakers interact with the room together — a mono measurement of one speaker at a time will give misleading results.

3 Run the Sweep

Click Start Measuring. Step well away from the microphone and stay perfectly quiet while the sweep plays. Any background noise — footsteps, closing doors, HVAC — can pollute the measurement and produce misleading scores.

REW will display the frequency response graph when the sweep is complete.

4 Export the Impulse Response

Once REW has generated the response graph, export the raw impulse response for Measurely to analyse:

  1. Go to File → Export → Export impulse response as WAV…
  2. Set the format to 32-bit float (or 24-bit) and Mono
  3. Save it somewhere convenient

Tip: If you're running REW 5.40 on the same machine as your browser, you can skip the export entirely — use the 1-Click Sync workflow above and Measurely pulls the data directly from REW.
